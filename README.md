<h1>AIM:</h1>

6.User Account Information

Step 1: To download Virtualbox

![1](https://github.com/Rajkiran276/OS-EX.2-INSTALLATION-OF-OS---CASE-STUDY/assets/147471453/896005f7-2eda-4882-8f6e-70b2a323dc3e)
